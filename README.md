# Kurocado Studio Platform

The **Kurocado Studio Platform** is the shared technical foundation used across all Kurocado Studio
products.  
It provides a consistent architecture, reusable services, design token infrastructure, and internal
tooling for building applications efficiently and at scale.

**[Read more about the platform.](https://kurocado-studio.github.io/platform/home.html)**

- Ship faster using shared patterns and tooling
- Grow a product ecosystem on a stable, scalable foundation
